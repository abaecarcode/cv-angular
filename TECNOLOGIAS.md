# Tecnologías Utilizadas en este Proyecto

Este Currículum Vitae interactivo ha sido desarrollado utilizando un stack tecnológico moderno, enfocado en el rendimiento, la escalabilidad y una experiencia de usuario fluida.

## 🏗️ Core & Framework
- **Angular 20**: Utilizando la última versión del framework de Google para la construcción de una Single Page Application (SPA) robusta.
- **TypeScript**: Para un desarrollo más seguro y tipado, mejorando la mantenibilidad del código.
- **Node.js**: Entorno de ejecución para las herramientas de desarrollo y compilación.

## 🎨 Diseño y Estilos
- **Bootstrap 5**: Framework CSS para la estructura base y componentes responsivos.
- **CSS3 Personalizado**: Implementación de estilos avanzados, incluyendo:
  - **Glassmorphism**: Efectos de desenfoque y transparencia para una estética moderna.
  - **Flexbox y CSS Grid**: Para layouts complejos y adaptables.
  - **Animaciones CSS**: Micro-interacciones y transiciones suaves para mejorar el UX.
- **Google Fonts**: Integración de tipografías modernas (Inter/Poppins) para mejorar la legibilidad.

## 🛠️ Herramientas de Desarrollo
- **Angular CLI**: Interfaz de línea de comandos para la gestión del ciclo de vida del proyecto.
- **NPM**: Gestor de paquetes para todas las dependencias del proyecto.
- **Git**: Control de versiones para el seguimiento del desarrollo.

## 📱 Características Destacadas
- **Responsive Design**: Totalmente adaptado a dispositivos móviles, tablets y ordenadores.
- **Navegación Fluida**: Uso del Router de Angular para cambios de sección instantáneos sin recarga de página.
- **Arquitectura Limpia**: Organización por páginas y componentes siguiendo las mejores prácticas de Angular.
